# 13.2 Architectural Alignment

Story Discovery shall remain fully aligned with the governing architectural artifacts of the Novel Development System.

Specifically, this Module shall conform to:

- NDS-002 — Engineering Doctrine
- NDS-003 — Constitution of the Novel Development System
- ARCH-000 — Repository Architecture
- ARCH-001 — System Architecture
- AG-001 — Architectural Governance
- MOD-000 — Module Standard
- REP-001 — Repository Engineering Standard

If a conflict exists between this Module and a governing artifact, the governing artifact shall take precedence until the conflict is formally resolved through the established architectural governance process.
